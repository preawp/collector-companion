
# Collectible Companion

A minimal React dashboard for tracking collectible item prices over time.

## Overview

Collectible Companion is a front-end prototype that allows users to:

- Track selected collectible items
- Store price snapshots over time
- Visualize price trends
- Prepare for future API integration

The goal is to observe value patterns and highlight fluctuations such as spikes or drops.
